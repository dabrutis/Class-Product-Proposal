Spotify Song IQ

David Abrutis

Quiz application according to personal or general Spotify songs/albums/artists selections.

Hosted through Azure, either a static or node.js website, and uses Spotify API for information about songs/albums/artists. Possibly using a database for high scores. HTML, CSS, and Javascript for languages, possible use of something like Bootstrap for its architecture.

The initial idea is to ask the user to guess what the most popular songs of an artist are and add together the points for a final score. Another idea would be to guess what album or artist a song is from.

Basic technical features are hosting a website, and making use of HTML, CSS, Javascript, and APIs to create the app.
