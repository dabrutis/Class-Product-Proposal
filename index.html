Spotify Song IQ

David Abrutis

Application Overview: Quiz application according to personal or general Spotify songs/albums/artists selections.

Technology Overview: Hosted through Azure, either a static or node.js website, and uses Spotify API for information about songs/albums/artists. Possibly using a database for high scores. HTML, CSS, and Javascript for languages, possible use of something like Bootstrap for its architecture.

Last Updated: The proposal was last updated on 11/10/2023.

Feature List: Features include the fetching of Spotify API JSON information to create a quiz-type application that will test the end userâ€™s knowledge of certain aspects of the music industry. Could be any number of unique quiz ideas, however the most prominent ones are ordering albums in chronological order, guessing the genre of a song, guessing the song based off lyrics, and so on. The idea is to implement at least one of the ideas, and hopefully more if time permits.

Future Feature List: If there was more time, use authentication to personalize the quizzes.
